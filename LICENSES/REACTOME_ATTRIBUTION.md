# Reactome attribution

This catalog can include pathway metadata derived from Reactome.

Pathway data derived from Reactome. Reactome data: CC0. Any Reactome pathway
layout or illustration adapted for this catalog must retain the applicable
Reactome CC BY 4.0 attribution and the Reactome release recorded by the build.

The release number and download time are written to `data/reactome/release.json`
when the opt-in Reactome build script is run.
